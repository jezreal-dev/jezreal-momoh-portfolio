# Skill: Technical Content Visual Production (Creative Director)

This skill defines the execution logic for the **Creative Director** role, utilizing Nano Banana to systematically generate graphics that match our developer-focused personal brand.

## Goal
To produce clean, professional visual assets that:
1.  **Avoid Clichés**: Never generate generic corporate stock photos (e.g., people in suits pointing at transparent screens).
2.  **Match the Aesthetic**: Adhere to the sleek, dark-mode developer aesthetic.
3.  **Enhance Clarity**: Elevate complex topics like AI architectures or security protocols through high-fidelity visual diagrams.

## Visual Standards

All generated assets must enforce these visual guidelines:

### 1. Color Palette
*   **Dominant Tones**: Deep charcoal gray (`#121212` to `#1E1E1E`).
*   **Accent Highlights**: Terminal green (`#00FF00`), neon cyan (`#00F0FF`), and electric blue (`#0047FF`).

### 2. Composition & Layout
*   **Isometric Systems**: 3D block-diagram layouts representing network infrastructure, data pipelines, or LLM agent sequences.
*   **Terminal & Code overlays**: Stylized console screens displaying clean syntax highlights or network packet capture outputs.
*   **Hardware Close-ups**: Modern circuit boards, glowing microchips, or keycaps in dim lighting.

### 3. Lighting & Tone
*   **Style**: High-contrast, cybertech lighting.
*   **Tone**: Mysterious, highly technical, and futuristic yet professional.

---

## Execution Instructions

When generating visuals for a new post or article, perform the following steps:
1.  **Read Context**: Scan the draft text to identify the core technical theme (e.g., "API Security", "RAG Pipeline").
2.  **Select Asset Type**: Choose between an *Isometric Architecture Diagram*, a *Stylized Terminal Screenshot*, or an *Abstract Cyber Concept*.
3.  **Formulate Prompt**: Build a prompt using the visual guidelines (specifying dark-mode, neon accents, and isometric composition).
4.  **Execute & Save**: Run the image generation and save the output in `outputs/visuals/`.
