# Brand Bible: Jezreal Momoh

This is the central reference document for all brand decisions, content creation, and positioning strategies.

---

## 1. Brand Core & Value Proposition
*   **Public Name**: Jezreal Momoh
*   **Location**: Ilorin, Nigeria
*   **Target Role**: AI & Software Full-Stack Engineer
*   **Tagline**: AI & Software Full-Stack Engineer | Documenting the Journey from Nigeria
*   **Core Value Proposition**: 
    > I help early-career African developers and builders understand what it actually looks like to build and deploy real, secure, AI-powered full-stack products—by sharing the raw, unfiltered journey of doing it myself, from Ilorin, Nigeria.

---

## 2. Background & Biography (Proof of Work)
*   **Academic**: Computer Science Graduate, Ambrose Alli University.
*   **Fellowship**: Software Engineering Fellow at Learn2Earn NG (Cohort 1).
*   **Founder**: Founder of **EduPilot** (an AI copilot for teachers, winner of the IIHDevBox hackathon).
*   **Approach**: Building in public, documenting real experiments, detailing errors/debugging, and focusing on full-stack AI development (React/Next.js frontend, Go & Python backend, and LLM/agentic systems).

---

## 3. Brand Persona & Voice
*   **Primary Archetype (60%)**: **The Tech Storyteller**. Focuses on story-first, personal, raw, and honest narratives of building in Nigeria (e.g., the boy with the note, building code in the bakery, NYSC teacher tools).
*   **Secondary Archetype (40%)**: **The Technical Educator**. Breaking down real full-stack technical concepts clearly (e.g., building React/Next.js interfaces, writing Go & Python APIs, securing LLM pipelines).
*   **Pronoun Use**: First-person singular ("I built...", "I learned...").
*   **Style Constraints**:
    *   No corporate filler language.
    *   No manufactured emotional build-up or fake drama.
    *   No em dashes (—) anywhere.
    *   No AI slop or clichés (e.g., "genuinely", "honestly", "straightforward").
    *   Formatting: Short, punchy paragraphs with line breaks, optimized for mobile LinkedIn reading.

---

## 4. Content Pillars
1.  **Building Full-Stack AI Products from Scratch**: Documenting the complete frontend-to-backend development journey of EduPilot (UI/UX, database schemas, API connectors).
2.  **Modern Full-Stack Engineering (Next.js/React + Go/Python)**: Technical deep-dives into modern web development, API design, concurrency, and hosting.
3.  **Deploying and Securing AI Systems**: How to implement RAG pipelines, manage agentic workflows, and secure the entire stack (front and back) against prompt injections and API vulnerabilities.
4.  **Personal Growth & The Human Side of Building**: Real-life stories of learning, grit, community, and the personal path to becoming an engineer in Nigeria.

---

## 5. Focus Revenue Model Roadmap
*   **Stage 1 (0–6 Months)**: Freelance full-stack engineering contracts, remote developer contract roles (e.g., micro-applications), and growing newsletter subscriber base.
*   **Stage 2 (6–18 Months)**: YouTube monetization, tech newsletter sponsorships, X revenue sharing, and brand deals with developer-first tools.
*   **Stage 3 (18+ Months)**: EduPilot SaaS subscriptions, technical courses (e.g., *Full-Stack AI Engineering*), consulting retainers, and paid developer communities.
